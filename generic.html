Image Classification Web App

A web app where users can upload images (e.g., handwritten digits or cats/dogs) and get predictions in real-time using a Convolutional Neural Network (CNN). The app is deployed using Streamlit or Flask for interactive demos.

  • Tech Stack: Python, TensorFlow / PyTorch, Streamlit / Flask
  • Key Features:
    • Real-time image predictions
    • Interactive web interface
    • Visual feedback for users
