AI Resume Chatbot (RAG)

This project converts my résumé into an intelligent chatbot that answers questions about my skills, experience, and projects. It uses embeddings with FAISS/OpenSearch to retrieve relevant information and an LLM (OpenAI/AWS Bedrock) to generate answers.

  • Tech Stack: Python, OpenAI API / AWS Bedrock, FAISS / OpenSearch, Next.js
  • Key Features:
    • Ask questions about my resume
    • Interactive web chat interface
    • Contextual answers from my resume
